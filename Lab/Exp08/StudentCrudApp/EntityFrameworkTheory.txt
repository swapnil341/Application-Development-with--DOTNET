Entity framework :-
 - Entity framework is a tool in dotnet that helps us work with data base
   using c# code instead of sql query.
 - less code, easy to development, faster development, easy to maintain.

types 
 - Entity framework :- older version, heavy , only for windows
 - Entity Framework Core :- new version, light, cross platform

 What is ORM:-
  - Object Relational Mapping
  - It maps C# classes to database tables and object to row
  - Ex 
  public class Student{
    public string Name {get;set;}
    public int Id {get;set;}
  } 

there are three types of coding approch 
 - Code first approch ( start with coding, create c# classes database is created automatically)
 - Database first approch (DB is already exist generate c# classes from DB)
 - Model first (design model first then DB created)

DB Context
 - DBContext is a main class that connects your application to the Database.
 - Ex
   public DbSet<Student> Strudents {get; set;}
 - DBContext is a bridge between application and database

Migration
 - Migration is used to create and update database structure automatically
 - Ex.
   public class Student{
    public string Name {get;set;}
    public int Id {get;set;}
    // new prop
    public string PRN {get;set;}
  } 

 - Migration will update database table.
